**PhaseCardView.swift:**

1. Add `@Environment(CommandRunnerService.self) private var commandRunner` to PhaseCardView.

2. Add a computed property `isRunning` that checks `commandRunner.activeRuns[project.path.path] != nil`.

3. Add a computed property `smartDefaultAction` that returns a tuple `(label: String, command: String, args: [String])`:
   - If `phasePlans.isEmpty` (no plans): return `("Plan", "claude", ["/gsd:plan-phase", "\(phase.number)"])`
   - If plans exist but not all done: return `("Execute", "claude", ["/gsd:execute-phase", "\(phase.number)"])`
   - If all plans done: return `("Verify", "claude", ["/gsd:verify-work", "\(phase.number)"])`

4. Add an action button row at the bottom of the content VStack (after the progress bar section), inside an HStack:
   - **When `isRunning`:** Show a red Cancel button (`Button("Cancel")`) with `.buttonStyle(.borderedProminent)` and `.tint(Theme.brightRed)`. On tap: `Task { await commandRunner.cancelRunningCommand(forProject: project.path) }`. Next to it, show `ProgressView().scaleEffect(0.6).frame(width: 14, height: 14)` and `ElapsedTimerView(startTime: run.startTime)` (get `run` from `commandRunner.activeRuns[project.path.path]!`).
   - **When not running:** Show the smart default button with `.buttonStyle(.borderedProminent)` and `.tint(Theme.accent)`. On tap: `commandRunner.enqueue(CommandRequest(command: action.command, arguments: action.args, projectPath: project.path, projectName: project.name))`. Next to it, show a `Menu` with `Image(systemName: "ellipsis")` label and `.menuStyle(.borderlessButton).fixedSize()`. Menu items: "Discuss" (`/gsd:discuss-phase`), "Plan" (`/gsd:plan-phase`), "Execute" (`/gsd:execute-phase`), "Verify" (`/gsd:verify-work`) — each with the phase number as argument. The ··· menu uses `Theme.textSecondary` for the ellipsis icon.
   - Add `Spacer()` between the button group and the ··· menu so buttons are left-aligned and menu is right-aligned.

5. Note: In views, `Task { }` works directly (no `_Concurrency.Task` prefix needed — the GSDMonitor.Task shadowing only affects Service files).

3. Add a computed property `isRunning` to PlanCard: `commandRunner.activeRuns[project.path.path] != nil`.

4. In PlanCard's header HStack (where plan number and status badge are), add between them:

On tap: `commandRunner.enqueue(CommandRequest(command: "claude", arguments: ["/gsd:execute-phase", "\(plan.phaseNumber)", "--plan", "\(plan.planNumber)"], projectPath: project.path, projectName: project.projectName))`. Disable if `isRunning`.

5. Update the `PlanCard` initializer call in `PhaseDetailView` to pass `project: project`.
