# Requirements: GSD Monitor

**Defined:** 2026-02-14
**Core Value:** Et live, grafisk overblik over alle GSD-projekter på tværs af ~/Developer — så du altid ved hvor du er, uden at åbne en terminal.

## v1.1 Requirements

Requirements for v1.1 Visual Overhaul. Each maps to roadmap phases.

### Theme Foundation

- [x] **THEME-01**: App bruger Gruvbox Dark farvepalet med alle 27 farver defineret som Color extensions (bg0-bg4, fg0-fg4, 16 accentfarver)
- [x] **THEME-02**: App kører altid i dark mode via NSApp.appearance — ignorerer system light/dark toggle
- [x] **THEME-03**: Alle hardcodede systemfarver (.blue, .green, .gray, .orange) er erstattet med semantiske Gruvbox-farver fra centraliseret tema-system
- [x] **THEME-04**: StatusBadge og PlanStatusBadge er konsolideret til én parametriseret komponent med Gruvbox-farver

### Progress Visualization

- [ ] **PROG-01**: Fase-kort viser animeret cirkulær progress-ring i stedet for lineær ProgressView
- [ ] **PROG-02**: Progress-ring animerer smooth ved første visning og ved statusændringer (ikke ved hver FSEvents-opdatering)
- [ ] **PROG-03**: Sidebar projekt-rækker viser mini progress-ring i stedet for lineær bar

### Visual Polish

- [ ] **VISL-01**: Fase-kort har gradient-header med Gruvbox-farver baseret på fase-status
- [ ] **VISL-02**: Sidebar projekt-ikoner bruger farvede SF Symbols med Gruvbox-palette baseret på projekt-status
- [ ] **VISL-03**: Empty states bruger themed ContentUnavailableView med relevante SF Symbols og forklarende tekst

### Dashboard

- [ ] **DASH-01**: Projekt-header viser stats-kort grid med: total faser, completion %, aktive faser, velocity
- [ ] **DASH-02**: Stats-kort bruger Gruvbox-accentfarver og har konsistent layout (ikon + tal + label)
- [ ] **DASH-03**: Milestone-tidslinje viser faser som forbundne noder i vertikal visning med status-farver
- [ ] **DASH-04**: Tidslinje-noder viser fase-navn, status og progress inline

## Out of Scope

| Feature | Reason |
|---------|--------|
| Brugerdefinerbare temaer | Tilføjer settings-kompleksitet, udvander branding, svært at vedligeholde kontrast |
| 3D/parallax effekter | Ikke macOS-nativt, distraherende, performance-kostbart |
| Glassmorphism/frosted glass | Svært at læse tekst, ikke tilgængeligt, overbrugt trend |
| Per-fase custom farver | Brugere definerer ikke fasetyper, bryder Gruvbox-konsistens |
| Cirkulær/radial tidslinje | Svær at læse kronologisk, dårlig pladsudnyttelse |
| Bouncing chart-animationer | Irriterende ved hyppige FSEvents-opdateringer |
