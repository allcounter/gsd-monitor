  <name>Task 1: Add gradient header to PhaseCardView</name>
  <files>GSDMonitor/Views/Dashboard/PhaseCardView.swift</files>
  <action>
Modify PhaseCardView to add a gradient background strip behind the header row (the HStack containing phase name and StatusBadge).

1. The existing `progressGradient` computed property already maps status to gradients for the progress bar. Add a new `headerGradient` computed property following the same pattern:
   - `.notStarted`: single-color LinearGradient with `Theme.fg4` (Theme.statusNotStarted — subtle gray for inactive phases, consistent with must_haves)
   - `.inProgress`: two-color LinearGradient from `Theme.yellow` to `Theme.brightYellow`
   - `.done`: two-color LinearGradient from `Theme.green` to `Theme.brightGreen`
   - All gradients: `startPoint: .leading, endPoint: .trailing`

2. Restructure the card body to wrap the header HStack in a ZStack with the gradient background:
   - The gradient sits behind the header content as a background layer
   - Apply `.opacity(0.25)` to the gradient so it is a subtle tint, NOT a full opaque block (this ensures text remains readable against bg1 card background blending through)
   - Header text should use `Theme.fg0` (bright white) for good contrast
   - The gradient area should cover just the header row with its padding, not the entire card
   - Use `.background()` modifier with the gradient + opacity on the header HStack, with a `RoundedRectangle(cornerRadius: 8)` clip shape so the gradient strip has rounded corners

3. Do NOT add animation on appearance (per research recommendation: start without, add later if requested).

4. Do NOT add `drawingGroup()` — premature optimization per research pitfalls.

5. Keep the gradient definition as a computed property (not inline in body) to avoid re-creation per research anti-patterns.
  </action>

Build the project: `cd . && xcodebuild -scheme GSDMonitor -destination 'platform=macOS' build 2>&1 | tail -5`

Verify no compile errors. Verify headerGradient computed property exists: `grep -n "headerGradient" GSDMonitor/Views/Dashboard/PhaseCardView.swift`
