Create `GSDMonitor/Views/Dashboard/` directory.

**PhaseCardView.swift:**
- Accepts `let phase: Phase` and `let project: Project`
- Layout: VStack(alignment: .leading, spacing: 12) with:
  - Header: HStack with "Phase {number}: {name}" (.headline) + Spacer + StatusBadge(status:)
  - Goal: Render phase.goal using `try? AttributedString(markdown: phase.goal)` with fallback to plain Text. Font .subheadline, .foregroundStyle(.secondary), lineLimit(3)
  - Requirements row: Only if phase.requirements is non-empty. HStack wrapping with "Requirements:" caption label, then ForEach over phase.requirements (id: \.self) rendering RequirementBadgeView for each
  - Progress bar: VStack with HStack("Progress" caption + Spacer + "{N}%" caption) above ProgressView(value:total:) with .linear style. Tint color: .gray for notStarted, .blue for inProgress, .green for done
- Completion calculation: Count plans matching this phase number from project.plans. completedCount / totalCount. If no plans, use 0.
- Card styling: .padding(), .background(Color(nsColor: .controlBackgroundColor)), .cornerRadius(12), .shadow(color: .black.opacity(0.1), radius: 4, x: 0, y: 2)
- Wrap entire card in Button(action: { showPhaseDetail = true }) with .buttonStyle(.plain)
- @SwiftUI.State private var showPhaseDetail = false
- .sheet(isPresented: $showPhaseDetail) presenting PhaseDetailView(phase:project:)

**RequirementBadgeView.swift:**
- Accepts `let requirementID: String` and `let project: Project`
- Computed property `var requirement: Requirement?` that looks up project.requirements?.first(where: { $0.id == requirementID })
- Button with .buttonStyle(.plain) showing:
  - Text(requirementID) with .font(.caption), .padding(.horizontal, 8), .padding(.vertical, 4), .background(badgeColor.opacity(0.15)), .foregroundColor(badgeColor), .cornerRadius(4)
- badgeColor: .blue for .active, .green for .validated, .orange for .deferred, .gray if requirement not found
- @SwiftUI.State private var showDetail = false
- .sheet(isPresented: $showDetail) presenting RequirementDetailSheet if requirement exists

**RequirementDetailSheet.swift:**
- Accepts `let requirement: Requirement` and `let project: Project`
- @Environment(\.dismiss) private var dismiss
- Computed property `var mappedPhases: [Phase]` filtering project.roadmap?.phases where requirement.mappedToPhases contains phase.number
- Computed property `var relatedPlans: [Plan]` filtering project.plans where plan.phaseNumber is in requirement.mappedToPhases (cross-reference for REQ-03)
- Layout:
  - Header: HStack with requirement.id (.title2 .bold) + requirement category (.subheadline .secondary) + Spacer + "Done" Button with .keyboardShortcut(.defaultAction) calling dismiss()
  - Divider
  - ScrollView with VStack(alignment: .leading, spacing: 20):
    - "Definition" section: headline + Text(requirement.description) with .textSelection(.enabled)
    - "Mapped to Phases" section: headline + ForEach mappedPhases showing "Phase {number}: {name}" with StatusBadge
    - "Related Plans" section (REQ-03): headline + ForEach relatedPlans showing "Plan {phaseNumber}-{planNumber}: {objective}" with plan status badge
    - "Status" section: headline + HStack(Circle fill + status text)
  - Status color: .blue active, .green validated, .orange deferred
- .frame(width: 500, height: 450)

**PhaseDetailView.swift:**

- @Environment(\.dismiss) private var dismiss
- Computed property `var phasePlans: [Plan]` filtering project.plans where plan.phaseNumber == phase.number, sorted by planNumber
- Layout:
  - Header: HStack with "Phase {number}: {name}" (.title2 .bold) + Spacer + StatusBadge + close Button("Done") with .keyboardShortcut(.defaultAction)
  - Divider
  - ScrollView with VStack(alignment: .leading, spacing: 20):
    - "Goal" section: headline + render phase.goal with AttributedString markdown (try? fallback)
    - "Requirements" section (if non-empty): headline + HStack wrapping ForEach phase.requirements with RequirementBadgeView
    - "Plans" section: headline + ForEach phasePlans showing plan cards:
      Each plan card: VStack in a rounded rect background showing:
        - HStack: "Plan {planNumber}" (.headline) + Spacer + plan status badge (similar to StatusBadge but for PlanStatus: .pending gray, .inProgress blue, .done green)
        - Text(plan.objective) (.subheadline .secondary) lineLimit(2)
        - "Tasks" section if plan.tasks non-empty: ForEach tasks showing:
          - HStack: task status icon (circle for pending, arrow.right for inProgress, checkmark.circle.fill for done) + Text(task.name) (.body)
    - If phasePlans is empty: Text("No plans found for this phase") .foregroundStyle(.secondary)
  - Plan card styling: .padding(12), .background(Color.secondary.opacity(0.05)), .cornerRadius(8)
- .frame(minWidth: 500, idealWidth: 600, minHeight: 400, idealHeight: 500)
