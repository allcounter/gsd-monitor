## Integration Check

**Score:** 6/6 Phase 16 exports properly wired

Phase 16 is well-integrated with the existing codebase. All service methods, popover UI, sidebar grouping, and UserDefaults persistence connect correctly.

### Minor Integration Issues

1. **groupedProjects sort gap** — `ProjectService.groupedProjects` does not sort ~/Developer first. `SidebarView` compensates, but `ContentView.swift` line 38 consumes it directly for auto-selection, making startup selection non-deterministic with multiple scan sources.

2. **Unreachable empty-state** — `SidebarView.swift` line 19 condition `scanSources.isEmpty` can never be true (defaults to `[~/Developer]`). Dead code.
